如何使用hexo + github建立自己的博客

GitHub配置

GitHub地址

1. 注册GitHub，这个不多说，不会自行百度

2. 注册成功后配置好SSH Key,方便后续电脑免密访问GitHub

3. 新建Repository,这里需要注意：

仓库的名称需要遵循如下规范：username.github.io
例如我的用户名是zhangsan，那这个仓库名就是：zhangsan.github.io

Hexo配置

1. 配置deploy插件

  • 在博客根目录下的_config.yml文件下，找到deploy那一行
    deploy:
  type: git                                                      # 设置发布类型，如git     
  repository: git@github.com: OUYANGSIHAI/ OUYANGSIHAI.github.io.git   # 设置repository对应的链接
  branch: master                                                 # 设置提交到的分支
  message: Site updated at {{ now("YYYY-MM-DD HH:mm:ss") }}      # 设置我们提交的信息

    2. 执行命令 hexo g

    3. 执行命令 hexo d

    4. 遇到不明错误 执行 hexo clean
